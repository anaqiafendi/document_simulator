# Bundled HDRIs

Three small CC0 HDRIs from [Poly Haven](https://polyhaven.com/hdris) used by the
v0.3 receipt synthesis pipeline (FDD #29). Bundled in-tree to keep the deploy
HF Spaces image self-contained — local users may drop additional `.hdr` files
into this directory and they will auto-populate the picker.

| File | Source | License |
|------|--------|---------|
| `office_warm.hdr` | Poly Haven `studio_small_03` (1k) | CC0 |
| `kitchen_bright.hdr` | Poly Haven `bathroom` (1k) | CC0 |
| `outdoor_overcast.hdr` | Poly Haven `symmetrical_garden_02` (1k) | CC0 |

Each HDRI ships with a pre-computed 128x128 PNG thumbnail (`<id>.thumbnail.png`)
generated by `scripts/build_hdri_thumbnails.py`. Thumbnails feed the v0.3d
React HDRI picker (`GET /api/receipt-synthesis/hdri-thumbnails`).

To regenerate thumbnails after adding/replacing an `.hdr`:

```bash
uv run python scripts/build_hdri_thumbnails.py
```
